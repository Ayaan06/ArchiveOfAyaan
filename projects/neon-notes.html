Neon Notes

A minimal note‑taking PWA with offline sync and keyboard‑first UX.

Live Source Code
  • PWA
  • TypeScript
  • IndexedDB
  • Service Worker

Overview

Neon Notes is a fast, offline‑capable note app designed for speed and focus. It caches assets, persists notes in IndexedDB, and provides a keyboard‑first editing workflow with instant search and tagging.

Features

  • Offline support via Service Worker (installable PWA)
  • Local persistence with IndexedDB and background sync
  • Instant search and tag filters
  • Keyboard shortcuts for all core actions

Challenges

Implementing conflict‑free syncing and ensuring a smooth offline/online transition required careful state handling and background sync strategies.
