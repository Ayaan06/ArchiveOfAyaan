To truly understand how computers think, I decided to build one from the ground up.

Overview

This project was my hands-on exploration of computer architecture and low-level programming, where I built a fully functional computer powered by the MOS Technology 6502 microprocessor. I designed, wired, and programmed a breadboard-based computer capable of running real code and producing output on an LCD display.

The Journey

I assembled the CPU, clock, and memory (EEPROM/SRAM) with a 1 MHz crystal. After verifying fetch/execute with machine-code loops, I wrote 6502 assembly to drive LEDs and a character LCD. I expanded I/O using a VIA for GPIO and a 6551 UART for RS-232, enabling me to interact from a PC terminal and load programs like WozMon and Microsoft BASIC.
