Grid Gallery

Responsive image gallery with lazy‑loading and color‑aware placeholders.

Live Source Code
  • JavaScript
  • IntersectionObserver
  • CSS Grid
  • Responsive

Overview

Grid Gallery balances performance and aesthetics: images lazy‑load as they enter the viewport and display color‑aware placeholders to reduce layout shifts and perceived loading time.

Features

  • CSS Grid layout across breakpoints
  • Lazy loading with IntersectionObserver
  • Blurhash/color placeholder previews
  • Keyboard navigation
