Overview

Built a modular Azure ML workspace that ingests DICOM data, performs preprocessing and augmentation, then serves PyTorch models as managed online endpoints. TensorFlow baselines were used for benchmarking before converging on a custom ResNet variant.

Workflow

A curated MLTable describes the dataset and powers automated runs. Each run logs metrics to Azure Application Insights, making it easy to compare experiments and detect drift.

Responsible AI

Incorporated fairness dashboards, bias slices, and interpretable Grad-CAM overlays so clinicians can trust the classification heatmaps and audit predictions over time.

Ops & Delivery

Model registry promotions trigger automated redeployments to production endpoints with rollback support, ensuring zero-downtime updates for hospital partners.

Key Takeaways

  • End-to-end ownership from data curation to monitoring.
  • Learned how to optimize GPU training pipelines in Azure.
  • Ship features that frontline healthcare teams can actually use.

Next Steps
